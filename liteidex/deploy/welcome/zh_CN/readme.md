LiteIDE X
=========

### 简介

_LiteIDE是一个简单，开源，跨平台的Go语言IDE._

* 版本: X20.1
* 作者: [七叶 (visualfc)](mailto:visualfc@gmail.com)


### 功能
* 核心功能
	* 系统环境管理
	* 编辑命令配置
	* 简单开放的调试系统
	* Kate格式语法高亮和自动完成
	* 可配置的语法WordApi列表
	* 基于Mime类型的系统
	* 完全插件支持
* Golang语言支持
	* 包浏览器
	* 结构视图和大纲
	* 文件浏览
	* [Gocode](https://github.com/nsf/gocode)代码自动完成支持
	* Go API检索
	* 跳转到声明
	* 当前代码表达式
	* Go playground
* Markdown支持
	* 实时预览和同步滚动
	* 可自定义CSS
	* 可导出为HTML和PDF文档
	* 批量转换/合并Markdown文档
* Golang Present支持
	* Present语法显示支持
	* 存盘时自动校验
	* 可导出为HTML文档

### 系统支持
* Windows x86 (32-bit or 64-bit) 
* Linux x86 (32-bit or 64-bit)
* MacOS X10.6
