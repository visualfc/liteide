LiteIDE X
=========

### Introduction
	LiteIDE is a simple, open source, cross-platform Go IDE.
	
### Base Features
* Mime type basis system
* System environment manage
* Build system manage
* Debug system simple and open
* Kate syntax and style scheme
* WordApi complete helper

### Golang Support
* Go Package Browser
* Golang Ast View
* Godoc Browser
* Gocode Helper
* Goapi Index
* Jump to Declaration
* Code Expression info
* Go Playground

### System
* Windows
* Linux 32  Linux 64
* MacOS X10.6
